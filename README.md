# Solar System

![ss](/docs/img/solar_system.gif)

A Three.js project showcasing an interactive solar system visualization built with [Three.js](https://threejs.org/)

![ss](/docs/img/ss_2.png)

## Overview

This project was created while following the [Zero to Mastery Three.js Crash Course](https://www.youtube.com/watch?v=KM64t3pA4fs&t=546s) by Jesse Zhou. It demonstrates core Three.js concepts including 3D rendering, camera controls, textures, and scene composition.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

## Project Structure

```
src/
├── main.ts           # Entry point
├── lib/
│   ├── camera.ts     # Camera configuration
│   ├── config.ts     # Global settings
│   └── planet.ts     # Planet objects
├── data/
│   └── planet-data.ts # Planet data
└── assets/
    └── textures/     # Texture files
```
